# buy_everything

這是一個 FB 直播輔助平臺，可以讓使用者在 FB 開完直播後，在我們的介面播放直播

並有商城介面，提供賣家販賣買家買產品

## 賣家
- 直播商品
- 新增/修改/刪除自有商品
- 推播商品
- 商品利潤呈現
- 信箱接收使用者下單訊息

## 買家
- 觀看直播
- 購買商品
- 新增/修改/刪除自有聯絡人
- 管理自我訂單
- 線上付款
- 信箱接收下單訊息

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
